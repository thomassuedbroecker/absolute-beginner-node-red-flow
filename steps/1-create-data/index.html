    Create data entries YouTube (2:40)¶

    In this part we create an inject and a cloudant out node.

    Background: In our situation we will use Node-RED on IBM Cloud with and Cloudant service instance, which contains a database to save the flows and more for our Node-RED instance. We use Cloud Foundry as our runtime for the Node-RED instance and with the existing Cloudant service binding, we easily can reuse our existing Cloudant service to create an absolute beginner database with a cloudant out node.
